# Employee Management System

A Full Stack Employee Management System built using the MERN Stack.

## Features

- Employee CRUD Operations
- Search Employees
- Filter by Department
- Sort Employees
- Pagination
- Export Employee Data to Excel
- Export Employee Data to PDF
- Dashboard Cards
- Dark Mode
- JWT Authentication
- Protected APIs
- Password Hashing using bcrypt
- MongoDB Database

---

## Tech Stack

### Frontend

- React.js
- Bootstrap
- Axios
- React Toastify
- SweetAlert2

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

## Installation

### Clone Repository

```bash
git clone https://github.com/dhanashrichule7-code/Emloyee-Management-System.git
```

### Install Client

```bash
cd client
npm install
npm run dev
```

### Install Server

```bash
cd server
npm install
npm run dev
```
